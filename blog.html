How I Made This Project

Jul 3, 2020

Now that Excuse Generator is going to be curated, Thingyman and I going to share how we made it. We first started off with a blank project. We went into the backgrounds, and created this:

Next, we added a button. It was the "Generate" button. We made it with the Text tool and the Rectangle tool in Vector mode. Our next step was to create the scripts. We created 3 lists, one for the names, one for the verb (destroyed, etc.), and one for the object. Then, I created this script for each.

We googled celebrities and put them in, and found synonyms for "destroyed". We also brainstormed objects.

When it receives pick (when the project starts or the generate button is clicked), it flips through the list. Each variable repeats for a different amount. The copy button uses a list that gets the variables and the background, and replaces the list. When it is clicked, it switches to “close”. It uses an if-then statement to decide what to do when it is clicked.

If you were wondering how the backgrounds change, that’s simple! I used a script that, when the flag is clicked or the generate button is pressed, sets the background to pick random (1) to (4). It also sets the brightness to -5 when touching the mouse, to achieve that effect.The script is below.
